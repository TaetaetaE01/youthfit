# PRD — 제품 개요 & 공통 컨텍스트

## 1. 제품 개요

### 제품명
YouthFit

### 한 줄 정의
흩어진 청년 정책 정보를 한곳에 모아, 쉬운 설명·적합도 판정·출처 기반 Q&A로 사용자가 자격 요건·준비사항·다음 행동을 빠르게 이해할 수 있도록 돕는 서비스

### 제품 포지셔닝
YouthFit은 공식 정책 포털(온라인청년센터, 정부24 등)을 **대체하지 않는다**. 정책을 더 쉽게 찾고 이해하도록 돕고, 최종 신청은 공식 채널로 연결하는 **보완형 탐색·해석 레이어**다.

| 역할 | YouthFit이 하는 것 | YouthFit이 하지 않는 것 |
|------|-------------------|----------------------|
| 탐색 | 정책 수집·정규화·검색 | 원천 포털을 완전히 대체 |
| 해석 | 쉬운 말 요약, AI 가이드 | 법적 효력 있는 자격 판정 |
| 연결 | 공식 신청 채널 안내 | 대리 신청·접수 |

### 핵심 사용자 가치
1. 관련 정책을 빠르게 찾을 수 있다.
2. 어려운 정책 문구를 쉬운 말로 이해할 수 있다.
3. 원문 공고를 전부 읽지 않아도 다음 행동을 알 수 있다.

## 2. 문제 정의

### 현재 상황
청년 정책은 중앙 부처, 지자체, 공공기관, 민간 위탁 기관 등 수십 곳에서 독립적으로 공고된다. 각 채널마다 공고 양식, 용어, 게시 주기가 달라 사용자가 정보를 종합적으로 파악하기 어렵다.

### 사용자가 겪는 문제

| 문제 | 상세 |
|------|------|
| **정보 분산** | 정책이 여러 공식·비공식 채널에 흩어져 있어 한곳에서 비교·탐색이 불가능하다 |
| **이해 장벽** | 행정 용어와 법률 문구가 개인의 실제 상황에 대입하기 어렵다 |
| **행동 포기** | 정책의 존재를 알아도 신청 경로가 불명확해 포기하는 경우가 많다 |
| **마감 누락** | 지원 일정을 직접 추적해야 해서 신청 기간을 놓치는 경우가 빈번하다 |

## 3. 타겟 사용자 & 페르소나

### 타겟 사용자 정의
- **연령대**: 만 19세 ~ 34세 (청년 정책 수혜 연령 기준)
- **핵심 타겟**: 만 24세 ~ 29세
- **대표 집단**: 사회 초년생, 취업 준비생, 최근 졸업자
- **공통 특성**: 소득·고용·주거·가구 상황이 자주 변하여 수혜 자격이 유동적

### 페르소나 A — 김민지 (26세, 취업 준비생)

| 항목 | 내용 |
|------|------|
| **배경** | 4년제 대학 졸업 후 1년째 취업 준비 중. 경기도 수원에서 부모님과 거주. 아르바이트로 월 80만 원 정도 수입이 있다. |
| **정책 경험** | 청년내일채움공제에 관심이 있지만, 온라인청년센터·고용24·지자체 홈페이지마다 안내가 조금씩 달라서 혼란스럽다. 한번 서류를 잘못 준비해서 접수가 반려된 적이 있다. |
| **동기** | "나한테 맞는 정책이 뭔지, 서류를 뭘 준비해야 하는지 한눈에 보고 싶다." |
| **Pain Point** | 정책 용어가 어렵다. "중위소득 150% 이하"가 본인에게 해당되는지 매번 검색해야 한다. 마감일을 놓친 적이 여러 번 있다. |
| **기대 행동** | 키워드 검색으로 정책을 찾고, 적합도 판정으로 자격 해당 여부를 빠르게 확인하고, 마감 알림을 설정한다. |
| **성공 기준** | 3분 이내에 자신에게 해당되는 정책을 찾고, 다음 행동(서류 준비 or 신청 링크)을 파악한다. |

### 페르소나 B — 이준혁 (29세, 사회 초년생 / 1년 차 직장인)

| 항목 | 내용 |
|------|------|
| **배경** | 중소기업에 취업한 지 8개월 차. 서울 관악구에서 월세로 독립 거주. 연봉 3,200만 원. 부모님 소득이 있어 가구 소득 기준이 애매하다. |
| **정책 경험** | 청년월세지원, 청년내일저축계좌에 관심이 있지만, 독립 가구 인정 기준을 확인하는 데만 30분 넘게 걸렸다. 결국 지원 자격이 되는지 확신을 못 해서 신청을 미루고 있다. |
| **동기** | "복잡한 조건을 내 상황에 맞춰서 알려주면 좋겠다. 공고 원문을 처음부터 읽기가 너무 부담스럽다." |
| **Pain Point** | 일하면서 정책을 비교할 시간이 없다. 여러 정책을 각각 별도 사이트에서 확인해야 한다. 정책 조건이 매년 바뀌는데 변경 사항을 추적하기 어렵다. |
| **기대 행동** | 관심 정책을 북마크하고, Q&A로 조건을 확인하고, 마감 전 이메일 알림을 받는다. |
| **성공 기준** | 점심시간 10분 안에 관심 정책 2~3개의 자격 조건을 확인하고, 해당 여부에 대한 가이드를 얻는다. |

### 페르소나 C — 박서연 (23세, 대학 재학생)

| 항목 | 내용 |
|------|------|
| **배경** | 지방 소재 대학교 3학년. 경북 구미에서 자취 중. 국가장학금을 받고 있고, 아르바이트로 월 50만 원 수입이 있다. 정책 제도 자체에 대한 이해가 적다. |
| **정책 경험** | "청년"이 붙은 정책이 있다는 것은 알지만, 재학생도 해당되는지, 어디서 신청하는지를 모른다. 친구의 SNS 공유로 처음 정책 정보를 접한 적이 있다. |
| **동기** | "나도 받을 수 있는 게 있는지 궁금하다. 근데 정책 사이트가 너무 딱딱하고 어디서 시작해야 할지 모르겠다." |
| **Pain Point** | 정책 포털의 진입 장벽이 높다. 검색어를 뭘로 해야 하는지 모른다. 정책 설명이 길고 어렵다. |
| **기대 행동** | 카테고리 탐색으로 정책을 둘러보고, 쉬운 요약을 읽고, Q&A로 궁금한 점을 질문한다. |
| **성공 기준** | 처음 접속 후 5분 안에 자신이 신청 가능한 정책이 있는지 여부를 파악한다. |
